## 管理员能做什么？

| 功能 | 说明 |
|------|------|
| ✅ 添加/编辑/删除项目 | 管理你的 vib coding 作品 |
| ✅ 上线/下线项目 | 控制用户能看到哪些项目 |
| ✅ 添加/编辑/删除文章 | 管理博客内容 |
| ✅ 发布/下架文章 | 控制用户能看到哪些文章 |
| ✅ 修改个人信息 | 更新你的介绍、社交媒体链接 |
| ✅ 修改密码 | 保护你的账号安全 |

## 工作流程

```
你（管理员）                    普通用户
    │                              │
    │  登录后台                    │
    │  添加/编辑项目               │
    │  点击"上线"                  │
    │  ─────────────────────────> │
    │                              │  刷新页面
    │                              │  看到新项目！
```
